# ☕ UG 茶飲 · 營運監控儀表板

> 全動態連鎖茶飲品牌營運監控 Dashboard

## 🔗 線上預覽
👉 **[https://johney8590-svg.github.io/ug-operations-dashboard](https://johney8590-svg.github.io/ug-operations-dashboard)**

## 功能特色
- 📑 **今日現況**：最新日期 KPI 卡片、各門市排行、渠道圓餅圖
- 📈 **趨勢分析**：整月走勢、週間分佈、天氣影響
- 📊 **門市對比**：4 張核心圖表
  - 圖表 A：本週走勢 vs 過去 4 週基線
  - 圖表 B：客單價異常偵測（雙 Y 軸）
  - 圖表 C：門市 Top / Bottom 排名
  - 圖表 D：渠道結構健康度（100% 堆疊）

## 技術棧
- HTML5 / CSS3 / Vanilla JavaScript
- [Chart.js 4.4](https://www.chartjs.org/)
- 模擬數據內建（含刻意製造異常場景）
